# WeSports Sporting Goods Website
CIST 2931 : Advanced Systems Project Management
Solange Alexandre-Petrie

This project was a team project to design and implement a Sporting Goods E-Commerce website.
The site needed to include the following functionality:

Customer login
Customer order review
Customer data retrieval 
Customer data updating
Customer shopping and checkout
Guest shopping and checkout
Order Fulfiller/Distributor login
Fulfiller/Distributor Order retrieval
Fulfiller/Distributor Order fulfillment

This project was completed with a team of 2 front-end developers and 3 back-end developers over a single semester.
